# TwoGather

TwoGather is a Flutter-based social app designed to help people connect in simple, meaningful ways.  
It lets users share posts, discover others, and interact through a clean, modern interface.

## Highlights
- Fast, responsive Flutter UI  
- User profiles, posts, and interactions  
- Scalable backend (e.g., Firebase)

## Getting Started
Install Flutter, clone the project, and run on your device or emulator.

More details will be added as the app grows.
